Demo Data

Follow below steps to import demo data of Easyfieds as it shows on the Easyfieds demo site on your system.

Folder Files

These are the main folders you have when you extract the project from zip file you received.

  1. Open Database Scripts > Demo Data folder. You have db-data-script.sql file and images directory in this folder. Sql script file contains sql queries for demo data and 'images' folder has images related to the items added via this script.
  2. Open db-data-script.sql file in your sql server and execute it on your database for Easyfieds. It will add categories, items and pictures etc in your database.
  3. After successfully adding products data via previous step, copy images from images folder to App.Web>Content>images location.
  4. Open your project in browser of your choice. It will have all the demo data as displayed in demo of Easyfieds.
